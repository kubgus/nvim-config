# nvim config

My personal NeoVim configuration files.

## prerequisites
- nvim 9.0+
- ripgrep
- packer

## features
- [x] theme
- [x] treesitter
- [x] telescope
- [ ] undotree
- [ ] lsp

# setup nvim
1. go to https://github.com/neovim/neovim/releases
1. download the latest release of nvim
1. extract the file in '/bin' using 'sudo tar xvzf ...'
1. 'ln -s .../bin/nvim ./nvim'
